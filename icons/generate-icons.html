Bookmark Organizer 图标生成器

预览当前 SVG 设计，并一键导出 PNG 图标文件

图标预览

自定义 SVG 转 PNG

如果你修改了 SVG 文件，可以在此选择文件转换为任意尺寸的 PNG。

使用方法

  • 直接点击「批量下载全部 PNG」，即可获取 icon16.png、icon48.png、icon128.png 三个文件。
  • 将下载的 PNG 文件放入扩展项目的 icons/ 目录，替换旧文件。
  • 如需修改图标设计，编辑对应的 icon16.svg、icon48.svg、icon128.svg 文件，然后使用上方的「自定义 SVG 转 PNG」功能重新生成。
提示：Chrome 扩展需要 PNG 格式图标，但 SVG 是矢量源文件，方便后续修改。建议保留 SVG 文件作为设计源稿。

命令行转换（ImageMagick）

如果你已安装 ImageMagick，也可以使用以下命令批量转换：

cd icons

# 转换三种尺寸的图标
magick -background none icon16.svg  -resize 16x16  icon16.png
magick -background none icon48.svg  -resize 48x48  icon48.png
magick -background none icon128.svg -resize 128x128 icon128.png
