# Bookmark Organizer

スマート、ローカル、プライバシー最重視 — ブラウザのブックマークを自動的に整理します。

ブックマークをスキャン、分類、重複削除、バックアップするブラウザ拡張機能です。すべての処理は完全にデバイス上で行われ、サーバーに何もアップロードされません。

## プロジェクトの位置づけ

Bookmark Organizer は、バニラ JavaScript と Manifest V3 で構築されたブラウザ拡張機能です。既存のブックマークマネージャーを置き換えることを目的としているのではなく、すでにあるブックマークを再び実用的に使えるようにすることを目指しています。

- **整理優先**: タイトル、URL、ドメインキーワードに基づいて自動的にカテゴリを提案します。カスタムルールはUIまたはJSONで追加できます。
- **クリーニング優先**: 完全な重複、正規化された重複（同じパス、異なるパラメータ）、類似した重複（同じドメイン、類似したタイトル）を検出します。
- **安全性優先**: スキャンのたびに自動バックアップを作成します。ローカルに直近10件のバックアップを保持し、ワンクリック復元と手動JSON/HTMLエクスポート/インポートに対応します。
- **プライバシー優先**: すべてのデータはブラウザストレージに保持されます。ネットワークリクエストなし、テレメトリなし、アカウント不要。
- **パーソナライゼーション優先**: 8つの組み込みスキン + カスタムスキンジェネレーター（任意の画像をアップロード）。完全なダークモードと5言語i18n対応。

> **Important**
>
> 拡張機能のデフォルト設定は一般的な用途向けに設計されています。インストール後は**設定**画面を開き、類似度閾値、自動バックアップ動作、テーマスキン、カスタム分類ルールを調整することをお勧めします。

### 方法1: ブラウザ拡張機能ストア

上記のストアリンクから直接インストールしてください。拡張機能は自動的に更新されます。

### 方法2: 開発者モード（テストまたは個人使用向け）

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/cheechang/BookmarkOrganizer.git
   cd BookmarkOrganizer
   ```

2. `icons/` ディレクトリに以下の正確なサイズの PNG ファイルが含まれていることを確認してください:
   - `icon16.png` — 16x16
   - `icon48.png` — 48x48
   - `icon128.png` — 128x128

   SVG のみの場合は、オンライン変換ツールを使用して PNG を生成してください。

3. ブラウザの拡張機能管理ページ（Edge の場合は `edge://extensions/`）を開き、右上隅の**開発者モード**を有効にします。

4. **パッケージ化されていない拡張機能を読み込む**をクリックし、プロジェクトのルートディレクトリを選択します。

5. ツールバーに拡張機能のアイコンが表示されます。クリックして使用を開始します。

## 主要機能

| モジュール | 機能 |
| --- | --- |
| スマート分類 | ブックマークをスキャンし、タイトル/URL/ドメインキーワードに基づいてフォルダを提案します。信頼度スコア（高/中/低）。UIまたは `rules/categories.json` でカスタムルールに対応。 |
| 重複検出 | 3つのタイプ: 完全一致（URLが同じ）、正規化された重複（同じパス、異なるパラメータ）、類似した重複（同じドメイン、類似したタイトル）。重み付け類似度アルゴリズム（タイトル60%、URL40%）。 |
| スマート選択 | 重複グループをインテリジェントに事前選択: ブックマークバーのコピーを保持し、残りはワンクリックでクリーンアップできるように自動的にチェックします。 |
| バックアップと復元 | スキャン前に自動バックアップ。直近10件を自動クリーンアップで保持。ワンクリック復元。JSONまたは標準Netscape HTMLへのエクスポートに対応。 |
| リンク切れチェック | HTTP HEAD/GETでタイムアウト付きでアクセス不可のブックマークを検出。404、5xx、タイムアウト、ネットワーク障害を特定。バックアップ統合付きの一括削除。 |
| クロスブラウザインポート/エクスポート | Chrome、Firefox、Edge、Safariから標準Netscape Bookmark HTMLをインポート。HTMLまたはJSONとしてエクスポート。マージまたは置換モード。 |
| テーマスキンシステム | 8つの組み込みスキン（デフォルト、ブラウザネイティブ、ミニマルビジネス、クラシックノスタルジア、ハイコントラストモノクロ、フロストガラス、ナチュラル低彩度、さらに4つのグラデーションテーマ）。任意のアップロード画像からカスタムスキンを生成。 |
| 完全なi18n | 5言語での完全なUI翻訳: 英語、簡体字中国語、スペイン語、日本語、ドイツ語。リアルタイム切り替えと即時再レンダリング。 |
| ダークモード | ワンクリックでライト/ダークテーマを切り替え。設定はブラウザストレージに永続化。すべてのスキンに専用のダークパレットが含まれています。 |

## 体験デザイン

Bookmark Organizer のインターフェースは「明確さ、効率性、視覚的な快適さ」を中心に構築されています。

- **デュアルインターフェース**: サイドバーナビゲーション付きの完全なスタンドアロンページ（`options.html`）、およびクイックアクセス用のコンパクトポップアップ（`popup.html`）。ツールバーアイコンをクリックすると、完全なページが直接開きます。
- **左サイドバーナビゲーション**: スキャン、分類、重複、リンク切れ、バックアップ、設定 — それぞれにカスタムSVGアイコンとアクティブ状態インジケーターがあります。
- **アダプティブスキン**: すべてのUI要素 — サイドバー、カード、リスト、フォーム、プログレスバー、フィルタータグ — は、選択されたスキンとライト/ダークモードにCSSカスタムプロパティを介して動的に応答します。
- **DPI自動適応**: CSSメディアクエリが100%、125%、150%、200%+の画面スケーリングを処理し、高DPI画面でのレイアウト崩れを防ぎます。
- **インラインアクション**: 一括選択、個別削除、展開/折りたたみ、ソート可能な列、フローティングスクロールアシストボタンで、インタラクションを手の届く範囲に保ちます。

## 技術スタック

| カテゴリ | 選択 |
| --- | --- |
| 言語 | バニラ JavaScript (ES2020+) |
| アーキテクチャ | ES Modules — バンドラーなしのモジュールインポート |
| マニフェスト | Manifest V3 (Chromium) / Manifest V3 + `background.scripts` (Firefox) |
| UI | 手書きの HTML + CSS、フレームワークなし |
| スタイル | CSSカスタムプロパティ（`--bo-*`）でスキンシステムを構築、`color-mix()` でアダプティブトーンを実現 |
| ストレージ | `chrome.storage.local` で設定、バックアップ、ログ、カスタムルールを保存 |
| ブックマークAPI | `chrome.bookmarks` で読み取り、作成、移動、削除、ツリー走査 |
| 類似度 | Levenshtein距離（編集距離）で重み付け平均タイトル/URLを比較 |
| i18n | `_locales/` で `__MSG_*__` プレースホルダーとランタイム `_t()` エンジン |
| アイコン | `currentColor` を使用したカスタムSVG線スタイルアイコンでテーマ適応 |
| CI/CD | GitHub Actions — Chromium + Firefox のクロスプラットフォームマトリックスビルド |

## プロジェクト構成

````bash
BookmarkOrganizer/
├── manifest.json              # Chromium拡張機能設定 (MV3)
├── manifest-firefox.json      # Firefox拡張機能設定 (MV3)
├── background.js              # Service Worker: インストールイベント、アイコンクリックハンドラー
├── options.html/css/js        # フルページのスタンドアロンインターフェース
├── popup.html/css/js          # コンパクトなポップアップインターフェース
├── i18n.js                    # ランタイム翻訳エンジン
├── theme-system.css           # スキンシステム: 8+スキン x ライト/ダーク変数
│
├── shared.js                  # 純粋なユーティリティ関数 (escapeHtml など)
├── bookmark-scanner.js        # ブックマークスキャン、分析、リンク切れ検出
├── category-manager.js        # スマート分類とバッチ移動操作
├── duplicate-detector.js      # 重複検出: 完全一致、正規化、類似
├── backup-manager.js          # バックアップ/復元、HTML/JSONインポート＆エクスポート
├── logger.js                  # クライアントサイドデバッグロギングフレームワーク
├── utils.js                   # レガシー共有ユーティリティ（後方互換性）
│
├── rules/
│   └── categories.json        # デフォルト分類キーワードルール
├── _locales/                  # i18nメッセージ: en, zh_CN, es, ja, de
├── icons/                     # 拡張機能アイコン (16px, 48px, 128px)
├── docs/
│   ├── screenshots/           # README用UIスクリーンショット
│   ├── DEVELOPMENT_LOG.md     # 開発履歴
│   └── STORE_LISTING.md       # ストア掲載用テキスト
├── .github/
│   └── workflows/
│       └── release.yml        # 自動ビルド＆リリースパイプライン
├── CHANGELOG.md               # バージョン履歴
├── PRIVACY_POLICY.md          # プライバシーポリシー（ストア提出に必要）
└── LICENSE                    # MITライセンス
````

## 免責事項

> **Caution**
>
> 1. この拡張機能はブラウザのブックマークを変更します。スキャンのたびに自動バックアップが作成されますが、初回使用前に手動でバックアップをエクスポートすることを強く推奨します。
> 2. すべてのデータ処理はブラウザ内でローカルに行われます。ブックマークデータはどのサーバーにもアップロードされません。
> 3. リンク切れチェッカーはブックマークのURLにHTTPリクエストを送信します。これがネットワーク環境と現地の規制に適合していることを確認してください。
> 4. データの消失や予期しない動作が発生した場合は、バックアップ管理ページからすぐに復元してください。
